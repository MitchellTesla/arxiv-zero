zero.middleware.auth module¶

Example middleware for decoding encrypted JWTs on each request.

class zero.middleware.auth.ExampleAuthMiddleware(app)[source]¶

Bases: arxiv.base.middleware.base.BaseMiddleware

An example of middleware to handle auth information on requests.

Before the request is handled by the application, the Authorization header is parsed for an encrypted JWT. If successfully decrypted, information about the user and their authorization scope is attached to the request.

This can be accessed in the application via flask.request.environ['auth']. If Authorization header was not included, or if the JWT could not be decrypted, then that value will be None.
